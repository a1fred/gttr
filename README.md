# GitLab timetracking reports
Generate time reports for your gitlab projects

## Install
Clone this repo and run 
```sh
python3 setup.py install
```

## Usage:
```sh
$ gttr https://gitlab.com/api/v4 mysecrettoken
```

## Test
```sh
python3 setup.py test
```
